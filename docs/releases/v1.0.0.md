## Vizo v1.0.0

### 新增
- 新增菜单栏“更新”菜单，支持手动检查新版本
- 新增“永不更新”开关，可一键启用或取消勾选
- 新增 GitHub Release 自动更新能力，发现新版本后可直接在应用内下载并安装

### 优化
- 缩略图右上角删除按钮改为鼠标悬停后再显示，界面更干净
- 检查更新时会弹出“正在检查更新”窗口，避免无反馈
- 检测到新版本时会显示本次更新内容，并提供“立即更新”和“取消”按钮
- 在 `win-unpacked` 解压版中手动检查更新时，改为提示需使用安装版，不再显示冗长英文报错

### 说明
- 自动更新依赖 GitHub Releases 发布的 `latest.yml`、安装包和 blockmap 文件
- 安装版会在启动后自动静默检查更新；勾选“永不更新”后将跳过启动检查，但仍可手动检查
