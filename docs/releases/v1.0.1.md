## Vizo v1.0.1

### 修复
- 将“关于 Vizo”和“检查更新”相关弹窗改为软件内自定义 UI，不再显示系统原生弹窗
- 修复手动检查更新时异常未正确回收的问题，GitHub 上未发布安装包或缺少 `latest.yml` 时会给出明确中文提示

- 更新内容弹窗支持在软件内滚动查看版本说明
